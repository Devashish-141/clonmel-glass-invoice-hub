# Adding the Clonmel Glass Logo

## Instructions:

1. Save the Clonmel Glass & Mirrors logo image as `clonmel-logo.png`
2. Place it in the `public` folder of this project
3. The logo will automatically appear in the sidebar header

## Logo Specifications:
- **File name:** `clonmel-logo.png`
- **Location:** `f:\glassone\clonmel-glass-invoice-hub-v2\public\clonmel-logo.png`
- **Recommended size:** 300-400px width (height will auto-adjust)
- **Format:** PNG with transparent background (preferred) or JPG

## Fallback Behavior:
If the logo image is not found or fails to load, the system will automatically fall back to showing the "CG" placeholder badge.
